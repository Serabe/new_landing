Where we are today

  • Our git implementation: go-git, covers the majority of the plumbing read operations and some of the main write operations.
  • Fetching and language & library usage analysis of every git repository on GitHub, BitBucket and self-hosted cgit repos.
  • Built the world's fastest WeightedMinHash & k-means clustering implementations.
  • Proved the relevance of features we extract by training neural networks that classify the origin of code
  • Shown that RNN's with memory can learn code from a specific project and learned their limitations.
  • Currently training neural networks on natural language use.

What is next?

  • Reaching full feature parity on go-git with libgit2.
  • Building a data pipeline to fetch and process all changes to public git repos on the Internet in close to real-time.
  • Moving our infrastructure to bare metal servers with a total storage of 1.4 Petabytes and GPU slots in each machine.
  • Finalizing our first trained models on all 6.6mm developers.
  • Implementing magnet loss into TensorFlow for 'Metric Learning with Adaptive Density Discrimination' applied to code style.
  • Build a platform for developers to use our technology to find the right teams and projects to join.

Committed to open source and its community
