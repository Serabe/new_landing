We have a Code Analysis tool that extracts different features from the open source code available on GitHub, detecting language usage, number of repositories, commits, as well as any code accounting for more than 150 common use frameworks and libraries, including Django, Flask or Unity, as well as database usage including SQL and NoSQL solutions.

All these features are then processed, giving us an overview of the developer's professional profile, and updated weekly with the feedback received from the developers we have contacted, as well as their ongoing contributions to repositories.

source{d} matching algorithms use the code features extracted by code analyzers to match engineers with our open job offers. The matching takes into account an estimation of the professional experience, which we model by creating internal ranks for each language and framework we detect.

Apart from the purely technical information, we also try to enrich our matching using information publicly available on professional networks such as LinkedIn. We place a lot of effort in extracting relevant information such as expected roles, salary or relocation preferences in order to only recommend relevant job offers.

Our matching algorithm runs independently for each job offer, providing us with a list of recommended profiles. We contact suggested engineers only after identifying that they have the right experience and background.

Our Developer Relations team answers any questions asked by the candidate. If the candidate is interested in your offer, we will elicit more specific information about their work experience, and how this relates to your offer.

Once these questions are answered, we set up a short call that gives us enough information to reach an educated conclusion on whether or not to push the candidate forward. The candidate is able to 'step out' at any stage of the process if the offer does not suit them.

Following a successful interview, we prepare a report summarizing their skills and highlighting their relevant experience for the position. This report is presented to you for review. If you would like to meet the candidate, we introduce you via email. We maintain contact with both the candidate and yourself in order to offer assistance where necessary.

If at any stage you decide to terminate a candidate’s process, we will inform them of the decision on your behalf.
